### Tasks description for Lesson 6:

#### Комментарии:

Задания к уроку 6 выполнены.

1. В задании 1 перегрузка сравнения "==" не сравнивает id счетов, так как они всегда разные (в этом их смысл) и результат был бы всегда "не равно". Зато как раз id используется для *GethashCode()*.

2. В задании 2 в основной прародитель *Figure* добавлены методы *Show()* И *Hide()*, чтобы можно было управлять полем visible.
За вывод всех полей отвечает расширяемый метод *Details()*. Метод *ToString()* просто добавляет к нему название фигуры. Кроме этого, согласно заданию реализован метод вывода на экран *PrintState()*, в который добавлено немного "красоты" (информация о фигуре выводится ее цветом).
